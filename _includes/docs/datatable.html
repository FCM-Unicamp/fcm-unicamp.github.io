Data table

Component para usar em tabelas, permitindo as funções de filtro, organização, live search entre outras

HTML

Exemplo de código base

<div class="table-responsive">
    <div id="data-table-basic_wrapper" class="dataTables_wrapper">
        <table id="data-table-basic" class="table table-striped dataTable" role="grid" aria-describedby="data-table-basic_info">
            <thead>
                <tr>
                    <th>Nome</th>
                    <th>Cargo</th>
                    <th>Local</th>
                    <th>Idade</th>
                    <th>Data inicial</th>
                    <th>Salário</th>
                </tr>
            </thead>
            <tfoot>
                <tr>
                    <th>Nome</th>
                    <th>Cargo</th>
                    <th>Local</th>
                    <th>Idade</th>
                    <th>Data inicial</th>
                    <th>Salário</th>
                </tr>
            </tfoot>
            <tbody>
                <tr>
                    <td>Tiger Nixon</td>
                    <td>System Architect</td>
                    <td>Edinburgh</td>
                    <td>61</td>
                    <td>2011/04/25</td>
                    <td>$320,800</td>
                </tr>
            </tbody>
        </table>
    </div>
</div>

CSS

Inclua os arquivos de css no header após o carregamento de todo o tema

<link rel="stylesheet" href="Resources/public/css/components/data-table.min.js">

Jquery

Inclua os arquivos de JS no final da página após o jquery

<link rel="stylesheet" href="Resources/public/js/components/data-table.min.js">

Inicie o plugin depois da inclusão do JS

$(document).ready(function() {
    $('#data-table-basic').DataTable(); 
}
